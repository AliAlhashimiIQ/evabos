# Build Resources

This directory contains build resources for electron-builder.

## Icon Files

Place your application icon files here:
- `icon.ico` - Windows icon (256x256 recommended)
- `icon.png` - macOS/Linux icon (512x512 recommended)

## Installer Scripts

- `installer.nsh` - Custom NSIS installer script (optional)

## Creating Icons

You can create icons using:
- Online tools: https://www.icoconverter.com/
- Image editing software (GIMP, Photoshop, etc.)
- Icon generators

For now, electron-builder will use a default icon if none is provided.
